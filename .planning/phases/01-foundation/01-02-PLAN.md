**A. Create `lib/schemas/auth.ts`:**
```typescript
import { z } from 'zod'

export const TokenPayloadSchema = z.object({
  role: z.enum(['captain', 'contributor', 'viewer']),
  userId: z.string().uuid(),
  userName: z.string().min(1),
})

export type TokenPayload = z.infer<typeof TokenPayloadSchema>

export const LoginSchema = z.object({
  username: z.string().min(1, 'Username is required').max(50),
  password: z.string().min(1, 'Password is required'),
})

export const RegisterSchema = z.object({
  inviteCode: z.string().min(1, 'Invite code is required'),
  username: z.string().min(3, 'Username must be at least 3 characters').max(30).regex(/^[a-z0-9_]+$/, 'Username must be lowercase letters, numbers, and underscores only'),
  displayName: z.string().min(1, 'Display name is required').max(50),
  password: z.string().min(8, 'Password must be at least 8 characters'),
})
```

**B. Create `lib/schemas/users.ts`:**
```typescript
import { z } from 'zod'

export const UpdateRoleSchema = z.object({
  role: z.enum(['captain', 'contributor', 'viewer']),
})

export const UpdateProfileSchema = z.object({
  displayName: z.string().min(1).max(50).optional(),
  currentPassword: z.string().min(1).optional(),
  newPassword: z.string().min(8).optional(),
}).refine(data => {
  if (data.newPassword && !data.currentPassword) return false
  return true
}, { message: 'Current password required to set new password' })

export const ResetPasswordSchema = z.object({
  temporaryPassword: z.string().min(8),
})
```

**A. Rewrite `app/api/auth/login/route.ts`:**
- POST handler: parse body with LoginSchema.safeParse. On failure return 400 with `{ error: 'Invalid input', details: parsed.error.flatten().fieldErrors }`.
- Query users table by username. If not found or is_active=false, return 401 `{ error: 'Invalid credentials' }`.
- bcrypt.compare password. If wrong, return 401.
- Update last_login_at to now().
- signToken with { role, userId: user.id, userName: user.display_name }.
- Set cookie: `tf_session`, httpOnly, secure, sameSite='lax', path='/', maxAge 7 days.
- Return 200 with `{ user: { id, username, displayName, role } }`.

**B. Rewrite `app/api/auth/register/route.ts`:**
- POST handler: parse with RegisterSchema.safeParse. On failure return 400 structured errors.
- Query app_config for invite_code. Compare with submitted inviteCode. If wrong, return 403 `{ error: 'Invalid invite code' }`.
- Check username uniqueness (query users). If taken, return 409 `{ error: 'Username already taken' }`.
- Hash password with bcrypt (cost 12).
- Insert user with role='viewer', is_active=true, is_seed=false.
- signToken and set cookie (same as login).
- Return 201 with user info.

**C. Rewrite `app/api/auth/logout/route.ts`:**
- POST handler: clear the tf_session cookie (set maxAge=0).
- Return 200 `{ success: true }`.

**D. Update `app/api/auth/me/route.ts`:**
- GET handler: getTokenPayload(req). If null, return 401.
