  <name>Task 1: YouTube OAuth endpoints + import route + Zod schemas</name>
  <files>app/api/youtube/auth/route.ts, app/api/youtube/callback/route.ts, app/api/youtube/import/route.ts, app/api/youtube/status/route.ts, lib/schemas/youtube.ts</files>
  <action>
1. Create `lib/schemas/youtube.ts` with Zod schemas:
   - `ImportResponseSchema` for the import endpoint response shape
   - No request body needed for import (POST with no body, just auth)

2. Create `app/api/youtube/auth/route.ts` — GET endpoint (captain-only):
   - Verify JWT via `getTokenPayload(req)`, reject non-captain with 403
   - Build Google OAuth2 authorization URL using `createOAuthClient()`:
     - scope: `https://www.googleapis.com/auth/youtube.readonly`
     - access_type: `offline`
     - prompt: `consent` (CRITICAL — ensures refresh token on re-auth, per Research Pitfall 1)
   - Return `NextResponse.redirect(authUrl)`

3. Create `app/api/youtube/callback/route.ts` — GET endpoint:
   - Extract `code` from URL search params
   - Exchange code for tokens using `oauth2Client.getToken(code)`
   - Call `storeTokens()` to persist access_token + refresh_token + expiry_date in app_config
   - Call `getChannelInfo(oauth2Client)` to fetch and store channel_id + uploads_playlist_id
   - Redirect to `/dashboard` with `?youtube=connected` query param for success feedback
   - On error, redirect to `/dashboard?youtube=error`

4. Create `app/api/youtube/import/route.ts` — POST endpoint (captain-only):
   - Verify JWT via `getTokenPayload(req)`, reject non-captain with 403
   - Check for minimum import interval: read `youtube_last_import` from app_config, reject if < 15 minutes ago (quota protection, per Research Pitfall 6)
   - Call `getAuthenticatedYouTube()` to get authenticated client
   - Read `youtube_uploads_playlist_id` from app_config
   - Call `youtube.playlistItems.list({ part: ['snippet', 'contentDetails'], playlistId, maxResults: 50 })`
   - For each video NOT already in session_videos (match by youtube_video_id):
     - Group by publish date (format: "Practice - MMM D, YYYY")
     - Create session if needed, insert session_video rows
   - Update `youtube_last_import` timestamp in app_config
   - Return JSON: `{ imported: number, sessions_created: number, skipped: number }`
   - Wrap in try-catch: if `googleapis` throws `invalid_grant`, return 401 with "YouTube authorization expired — reconnect"

5. Create `app/api/youtube/status/route.ts` — GET endpoint (auth required, any role):
   - Check if `youtube_oauth_tokens` exists in app_config
   - If yes, return `{ connected: true, channelId }` (read from app_config)
   - If no, return `{ connected: false }`
